# Tudo-mais-delivery 

Esta é a minha contribuição para o desenvolvimento do aplicativo **Tudo-mais-delivery**. Este repositório contém especificamente a minha parte do trabalho, que engloba as telas de fluxo de checkout, acompanhamento de pedidos e gerenciamento de entregas.

O código foi desenvolvido utilizando **React Native** com o ecossistema **Expo**.

---

## 📂 Organização dos Arquivos desta Contribuição

Como publiquei esta parte do trabalho arquivo por arquivo, eles estão localizados na raiz deste repositório. No entanto, para que o aplicativo funcione corretamente, eles devem ser integrados seguindo a estrutura abaixo:

*   `App.tsx` e `logo.png` devem ficar localizados na **raiz** do seu projeto React Native.
*   Os arquivos de telas (`Acompanhamento.js`, `ConfirmarEntrega.js`, `EditarLocal.js`, `Inicio.js` e `PedidoFinalizado.js`) devem ser colocados dentro da pasta **`/screens/`**.
